An open-source remote desktop application, the open source TeamViewer alternative.

In order for a remote device to control your Android device via mouse or touch, you need to allow IrisRemote to use the "Accessibility" service, IrisRemote uses AccessibilityService API to implement Android remote control.

In addition to remote control, you can also transfer files between Android devices and PCs easily with IrisRemote.

You have full control of your data, with no concerns about security. 

Please download and install desktop version from: https://remote.irissecurities.com/, then you can access and control your desktop from your mobile, or control your mobile from desktop.
